# Security Policy

## Supported Versions

Security fixes are currently handled for the latest public AxumGeez release.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Email: support@adischat.com

Include:

- A clear description of the issue
- Steps to reproduce
- Affected version
- Any relevant logs or screenshots

We will review reports and respond as soon as possible.

## Scope

Security-sensitive areas include:

- Global Windows input handling
- Licensing and device validation
- Installer behavior
- Website download links
- Update and release signing process

AxumGeez does not collect typed content.
