# AxumGeez Website

Modern landing website for AxumGeez, built with Next.js 14 App Router, React, TypeScript, and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run lint
npm run build
npm run start
```

## Download File

Place the Windows installer at:

```text
public/downloads/AxumGeezSetup.exe
```

Until the installer is uploaded, the Download page shows a Coming soon note.
