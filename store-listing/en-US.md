# Axum Geez

## Product Name

Axum Geez

## Short Description

Modern Amharic (Ge'ez) typing software for Windows.

## Full Description

Axum Geez is a professional Windows typing application that helps users type Amharic and Ge'ez characters across Windows apps including Word, Excel, browsers, Notepad, messaging apps, and code editors.

## Features

- Type Amharic anywhere
- Complete Ethiopic Fidel support
- Ge'ez number support
- Fast phonetic keyboard layout
- System tray controls
- Brana font support
- Light and dark themes
- Offline typing
- Windows 10 and Windows 11 support
