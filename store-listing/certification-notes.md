# Microsoft Store Certification Notes

Axum Geez is a Windows desktop application for Amharic typing.

## Testing

1. Install using the silent command:

```text
/VERYSILENT /SUPPRESSMSGBOXES /NORESTART /SP-
```

2. Launch Axum Geez from Start Menu.
3. Toggle Amharic mode using Ctrl + Space or tray icon.
4. Open Notepad, Word, Excel, or browser text field.
5. Type sample: `selam` -> `ሰላም`.
6. Open Settings and Keyboard Layout from tray menu.

The application runs in the system tray to provide typing functionality.

It does not install drivers, NT services, browser extensions, or bundled software.

It does not collect typed text.
