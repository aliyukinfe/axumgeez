# AxumGeez Open Source

AxumGeez is open source under the MIT License.

## Repository

https://github.com/aliyukinfe/axumgeez

## What Is Included

- Next.js website source
- Windows desktop app source
- Transliteration engine
- Keyboard layout data
- Unit tests
- Installer script
- Documentation

## What Is Not Included

- Private code-signing certificates
- Cloud service secrets
- Generated build artifacts
- Local development settings

## Code Signing

The public source code is available so AxumGeez can qualify for open-source code-signing programs where accepted.

Open-source signing providers may still require review, project history, public repository visibility, release reproducibility, and publisher identity checks.
